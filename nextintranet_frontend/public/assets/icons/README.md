# Custom Category Icons

Place your custom category icons in this directory.

## Usage

To use a custom icon for a category, use the prefix `img:` followed by the filename.

For example:
- `img:resistor.png`
- `img:capacitor.svg`
- `img:ic-chip.jpg`

The image will be loaded from `/assets/icons/{filename}`

## Supported formats
- PNG
- SVG
- JPG/JPEG
- GIF
- WebP
